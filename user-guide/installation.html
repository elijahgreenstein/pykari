Pykari: Static Sites in a Flash

Installation

Pykari depends on two plugin extensions for Markdown-It-Py: Heading, MD and Highlight-It. Follow the installation instructions for each plugin. Then, clone the Pykari repository, change into the repository directory, and install Pykari with pip:

pip install .

pip will install Pykari, as well as the following additional dependencies (if not already installed):

  • Jinja
  • Markdown-It-Py
  • Markdown-It-Py Plugin Extensions
  • PyYAML
